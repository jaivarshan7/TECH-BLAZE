Description

"TRACE THE LINKS, FIND THE ROOT CAUSE"

Connection is an interactive challenge designed to enhance technical reasoning, pattern recognition, and creative problem-solving. The goal is to identify that hidden connection as quickly and accurately as possible.

Event Structure
  • Participation: Maximum of 2-4 participants per team.
Rounds

Round 1: The Debug Log (Clue Decoding)

Objective: Decode abstract symbols and snippets to identify specific technical terms.

Format: Visual puzzles using emojis and icons, Code snippet analysis.

Scoring: Correct clue decoded: +5 points.

General Rules:

  • Participants face timed challenges at each round.
  • Each round builds on the previous.
  • Negative marking is active: Incorrect guesses will result in point deductions.

Round 2: The Link Layer

Objective: Analyze multiple clues and identify the single hidden connection that links them together, testing participants’ pattern recognition and conceptual understanding.

Format:

  • Visual and textual clue sets (images, emojis, symbols, short code snippets, keywords).
  • Each set contains 4–6 clues connected by one underlying technical theme.
  • Teams must identify the common link and justify their reasoning.

Round 3: The Core Connection

Objective: Uncover a deep, meta-level technical connection by synthesizing insights from multiple layered clue-sets, using reverse reasoning, abstraction, and elimination, to identify the final unified system.

Format:

  • Teams receive 3-5 layered mini sets.
  • Each mini set contains visual symbols/emojis, short code fragments/logs with missing context.
  • Each mini set contains a property, behaviour, or constraint (Not direct keywords).
  • No mini set is solvable independently.
