CRACK THE LOGIC
Format
  • Category: Reverse Coding
  • Team Size: Maximum 2 participants
  • Rounds: 3 Rounds
  • Languages: C, Java, Python
Rules and Regulations
  • Only registered participants will be allowed to compete.
  • Involvement in malpractice will lead to immediate disqualification.
  • External devices are not allowed.
  • Lab facility will be provided by the Institution.

Round 1: The Logic Decipher (Written Round)

Type: Discussion Q/A | Mode: Offline (no coding, only reasoning or formulas)

Objective: Test participant’s analytical and logical thinking—they’ll be given outputs, patterns, or code snippets and must write the logic, formula, or algorithm (not actual code).

Round 2 – Debug the Logic (Error-Finding Round)

Type: Written (pen & paper)

Objective: Identify logical and syntactical errors in the given code snippets and explain how to correct them — without running the code.

Round 3 – Code the Mystery (Final Round)

Mode: Computer (C / Java / Python)

Objective: The ultimate reverse coding challenge — participants are given sample outputs and must write a working program that reproduces them without being told the logic. They must reverse-engineer the problem, discover the hidden pattern, and implement a complete solution.
