Format
  • Type: Decode SQL Clue
  • Team Size: Teams must have 2 or 4 members.
  • Rounds: 2 Rounds
  • Mode: Offline
Rules and Regulations
  • Only one person codes at a time — strict relay mode.
  • Systems and required software will be provided.
  • Teams must only use HTML, CSS and JavaScript for building the web application.

ROUND 1 – Prelims

  • Activity Type:
  • Mode: Paper-Based Objective Test

The first round is a paper-based objective test consisting of MCQs, match-the-following, fill-in-the-blanks, simple calculations, and data interpretation questions. This round focuses on speed, accuracy, and basic analytical skills to efficiently shortlist participants.

ROUND 2 – Data visualization

  • Task: Design visual representations (charts, dashboards, or storyboards).
  • Tools Allowed: Excel, Tableau, Power BI, Python (pandas, numpy, matplotlib,and seaborn)
  • Mode: Computer-based

Participants now move to their clue board—transforming raw data into meaningful visuals. Participants will be given a dataset and must create clear, creative, and accurate visualizations that uncover hidden insights.

ROUND 3 – Prediction & Presentation Round

  • Task: Perform analysis and present findings.
  • Mode: Computer-based.

In this final round, teams will present the data visualizations and insights created in the previous round. They must clearly explain their analysis and highlight key patterns and trends. The presentation will be evaluated by the judges based on clarity and understanding.

The judges and event coordinator's decision would be the final.
